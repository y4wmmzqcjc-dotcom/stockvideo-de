Einrichtungs-Anleitung

So richtest du Mollie ein

  1. Erstelle ein Konto auf mollie.com
  2. Gehe zu Entwickler → API-Schlussel
  3. Kopiere deinen Live-API-Schlussel (beginnt mit live_)
  4. Fuge die Webhook-URL oben in dein Mollie-Dashboard ein
  5. Aktiviere die gewunschten Zahlungsmethoden in Mollie
  6. Schalte den Modus auf Live um
Fur eine vollstandige Mollie-Integration benotigt die Website eine serverside Implementierung (z.B. Cloudflare Worker). Spreche deinen Entwickler an.

So funktioniert die Veroffentlichung

  1. Videos im Admin-Panel bearbeiten und speichern
  2. "Veroffentlichen" klicken
  3. Das Admin-Panel schreibt videos.json direkt per GitHub-API ins Repo
  4. Cloudflare Pages erkennt die Anderung und baut die Website neu (~20 Sek.)
  5. Die neue Videoliste ist live auf stockvideo.de
